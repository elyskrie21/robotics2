Arduino FM Radio Project

What we will be creating

In this tutorial, we are going to build a simple FM Radio receiver on a breadboard.

Materials

Since we will be creating a complicated project, we are going to need some cheap stuff

  • Arduino Uno
  • FM Radio Module
  • Nokia 5110 LCD
  • 2 * 10K Potentiometer
  • Audio Amplifier
  • 3W Speaker
  • Large Breadboard
  • Audio Cable
  • FM Transmitter

Creating the FM Radio

Nokia 5110 LCD

This is the screen we are going to use and its a bit complicated so just watch this video to set it up.

TEA5767 FM Radio Reciever

This is kina important cause we are making a radio.

The chip uses the I2C interface, so it is straightforward to use with Arduino. On the module, we can also find an audio amplifier. Unfortunately, this is a headphone amplifier so we cannot connect a speaker to the output of the module directly since the audio signal is very weak. We need to amplify it first. Also, there is no way to control the volume that comes out of the module with software, so we are going to use a potentiometer later on.
